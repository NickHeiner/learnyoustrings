Write a function that returns true if a string is a palindrome, and false otherwise.

Example
    yourSolution('foobar')
    // ==> false

    yourSolution('bbb')
    // ==> true

    yourSolution('abba')
    // ==> true

    yourSolution('amanaplanacanalpanama')
    // ==> true

----------------------------------------------------------------------
HINTS:

Strings are immutable.

    var str = 'asdf';
    str[0] = 'z'; // str === 'asdf'
                  // because strings are immutable, any assignment into them is ignored.

String.slice is a nice method for cutting up strings:

    'abcd'.slice(1) === 'bcd'
    'abcd'.slice(2) === 'cd'
    'fff'.slice(9001) === ''

If str[x] is the x-th character of str counting from the beginning of the string,
then str[str.length - 1 - x] is the x-th character of str counting from the end of the string.

    var str = 'fzobar',
        x = 2;

    str[x] // o
    str[str.length - 1 - x] // b
